# EAT-PICK 영양 랭킹

식품의약품안전처 음식·가공식품 엑셀 DB를 이용해 칼로리, 단백질, 당류 순위를 보여주는 로컬 웹앱입니다.

## 실행

터미널에서 아래 명령을 실행합니다.

```bash
cd "/Users/topjh/Desktop/codex office/eat-pick-dashboard"
./start.sh
```

처음 실행할 때 패키지 설치와 약 31만 건의 데이터 변환을 수행하므로 몇 분 정도 걸릴 수 있습니다. 준비가 끝나면 브라우저에서 <http://127.0.0.1:5173>을 엽니다. 종료는 터미널에서 `Ctrl+C`를 누릅니다.

원본 엑셀은 `data/raw/`에 포함되어 있으며 Git LFS로 관리됩니다. 저장소를 복제할 때 Git LFS가 설치되어 있어야 원본 파일도 함께 내려받을 수 있습니다.

## 데이터 갱신

새 엑셀 파일로 다시 만들려면 `data/foods.db`를 다른 곳에 보관하거나 삭제한 뒤 `./setup.sh`를 실행합니다. 원본 엑셀은 읽기만 하며 수정하지 않습니다.

## 비교 기준

- 원본의 `영양성분함량기준량`이 g 단위로 명시된 행을 100g 기준으로 환산합니다.
- ml, 1회 제공량처럼 무게 환산이 불가능한 행은 순위에서 제외합니다.
- 낮을수록 좋은 지표: 칼로리, 당류
- 높을수록 좋은 지표: 단백질
- 빈 영양성분은 해당 지표의 순위에서 제외합니다.
